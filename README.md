WebMail Notifier Linux driver (Dream Cheeky)
============================================

Installation
------------
dependency libusb++-dev
    cd src
    make
    sudo make install

Usage
-----

 * Switch the red light on `usblamp red`
 * Switch to green light on (hexcode) `usblamp #00FF00`
 * Turn the lamp off `usblamp off`
