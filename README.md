WebMail Notifier Linux driver (Dream Cheeky)
============================================

Installation
------------
dependency libusb++-dev
    cd src
    make
    sudo make install

Usage
-----

 * Switch the red light on `sudo usblamp red`
 * Switch to green light on (hexcode) `sudo usblamp #00FF00`
 * Turn the lamp off `sudo usblamp off`
